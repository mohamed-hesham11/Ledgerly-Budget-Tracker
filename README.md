# Ledgerly-Budget-Tracker
Ledgerly is a sleek, single-page personal finance application built entirely with vanilla HTML5, CSS3, and ES6+ JavaScript. Designed with a premium dark-mode FinTech aesthetic, it provides users with a distraction-free environment to log transactions and monitor their net balance.

Key Features:

Zero-Dependency Architecture: Built completely without external frameworks or libraries for lightning-fast performance.

Dynamic Data Visualization: Utilizes native CSS Flexbox injection to generate real-time expense breakdown bars based on category spending.

Persistent Storage: Hydrates state automatically using browser localStorage to ensure financial data is never lost between sessions.

Responsive UI: Centralized 700px single-column layout optimized for readability and quick mobile-to-desktop scaling.
